Firebase Setup Required

To enable Google login & real-time sync, paste your Firebase configuration below.

📋 How to get Firebase config (5 minutes):
1. Go to console.firebase.google.com
2. Click "Create Project" → name it "tipi-puja"
3. Go to Project Settings → General
4. Scroll to "Your apps" → click Web icon (</>)
5. Copy the firebaseConfig object
6. Enable Google Sign-In in Authentication
7. Enable Realtime Database → start in test mode
8. ⚠️ Realtime Database → Rules tab → Replace with: 
{
  "rules": {
    "users": {
      ".read": "auth != null",
      "$uid": { ".write": "auth != null" }
    },
    "settings": {
      ".read": "auth != null",
      ".write": "auth != null"
    },
    "puja_data": {
      ".read": "auth != null",
      ".write": "auth != null"
    }
  }
}
Then click Publish. This blocks all unauthenticated access.

Your data stays in your own Firebase project. Anthropic cannot access it.
